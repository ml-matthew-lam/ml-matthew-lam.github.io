Deblurring Network Ablation Study

An empirical evaluation of architectural components in Deep Image Deblurring.

Project Overview

This project systematically evaluates the impact of core architectural modifications on a baseline Encoder-Decoder network designed for dynamic scene deblurring. By isolating components like multi-scale feature aggregation, residual dense blocks, and attention mechanisms, we analyze the trade-offs between computational complexity and reconstruction quality.

Key Takeaways

Attention Efficiency: Integrating coordinate attention yielded a +1.38 dB boost in PSNR while only expanding the parameter footprint by ~6%, proving highly cost-effective for spatial blur reconstruction.

Error Reduction: Combining both residual connections and channel-wise attention successfully neutralized high-frequency ringing artifacts that the baseline model failed to resolve.
